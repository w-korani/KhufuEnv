# KhufuEnv

- single-thread process.
- does not need big compuataional resources.
- does a simple job.

## Installation

- git clone https://github.com/w-korani/KhufuEnv
- cd KhufuEnv_main
- sudo ./installer
- echo "source /etc/KhufuEnv/call.sh"  >>  ~/.bashrc
- . ~/.bashrc
